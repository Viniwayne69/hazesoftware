# Portfólio Vinícius Ribeiro

Landing page profissional de portfólio pessoal para Vinícius Ribeiro, criada com HTML, CSS e JavaScript puros.

## Estrutura

```text
portfolio-vinicius/
├── index.html
├── css/
│   ├── style.css
│   └── responsive.css
├── js/
│   └── script.js
├── assets/
│   ├── img/
│   └── icons/
└── README.md
```

## Como visualizar

Abra o arquivo `index.html` no navegador ou rode um servidor local simples dentro da pasta:

```bash
python -m http.server 8000
```

Depois acesse `http://localhost:8000`.

## Imagens

A página já funciona com mockups criados em CSS. Quando quiser substituir por imagens reais, adicione os arquivos abaixo em `assets/img/` e ajuste as referências no HTML ou no CSS:

```text
foto-vinicius.png
projeto-agenda-explosiva.png
projeto-hazelink.png
projeto-sdr-infinity.png
projeto-barbearia.png
projeto-manicure.png
mockup-hero.png
```

## Recursos incluídos

Menu mobile funcional, scroll suave, animações de entrada ao rolar a página, cards com hover premium, mockups visuais em CSS, foto real no hero com efeito ProfileCard adaptado, fundo Galaxy em WebGL puro e estrutura responsiva para desktop, tablet e celular.
